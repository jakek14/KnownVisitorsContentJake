## Post

Garry Tan went on stage at SXSW and said he's sleeping 4 hours a night because he can't stop AI coding.

He called it "cyber psychosis."

His Claude Code setup got 20,000 GitHub stars in two days. Half the internet called it "god mode." The other half said it was "just markdown files with prompts."

Both camps are right. The argument doesn't matter.

Here's what actually happened: the CEO of Y Combinator — not an engineer by trade — is shipping 10,000 lines of code a week.

The tools don't care about your job title anymore.

A year ago, "I want to build X" meant "I need to hire someone." Now it means "let me start prompting."

That shift is bigger than any one tool.

## Context

YC CEO Garry Tan open-sourced "gstack" on March 12, 2026 — a Claude Code configuration using 15 specialized markdown skill files to simulate an entire engineering org (CEO review, QA, release manager, etc.). The repo exploded: 20,000+ GitHub stars in days, 2,200+ forks. At SXSW, Tan confessed to sleeping 4 hours a night from pure AI-coding excitement, coining "cyber psychosis." The internet split: some called it "god mode," others dismissed it as fancy prompts.

The post captures a genuine founder take on this moment — not hype, not backlash, but the real underlying signal: the friction between having an idea and building a product just collapsed.
